# Interactive Calendar Component

A modern interactive calendar built with React featuring date selection, notes CRUD, and a responsive dark UI.

## 🚀 Live Demo
<your-vercel-link>


## ✨ Features
- Single & range date selection  
- Add, edit, delete notes  
- Notes stored in localStorage  
- Toggle selection (click again to unselect)  
- Month navigation  
- Live clock  
- Responsive layout  
- Dark UI  

## 🛠 Tech Stack
- React (Vite)  
- Tailwind CSS  
- date-fns
